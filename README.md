# 🏥 医院查询系统

一个功能完整的医院查询和管理系统，使用纯 HTML、CSS 和 JavaScript 开发。

## 功能特性

### ✨ 核心功能
- **查询医院** - 通过医院名称或城市快速搜索
- **按条件过滤** - 支持按城市和医院等级过滤
- **查看详情** - 点击查看每家医院的完整信息
- **添加医院** - 新增医院信息到系统
- **编辑医院** - 修改已有医院的信息
- **删除医院** - 移除不需要的医院记录
- **数据持久化** - 使用 LocalStorage 本地存储数据

### 🎨 用户界面
- 现代化的设计风格，采用紫色渐变背景
- 响应式布局，支持桌面和移动设备
- 流畅的动画和过渡效果
- 直观的操作流程和用户反馈

## 快速开始

1. 克隆或下载项目到本地
2. 在浏览器中打开 `index.html` 文件
3. 系统会自动加载内置的示例医院数据

## 使用说明

### 搜索医院
1. 在搜索框中输入医院名称或城市名
2. 点击"搜索"按钮或按 Enter 键
3. 系统会显示匹配的医院

### 过滤医院
- 使用城市下拉列表按城市过滤
- 使用等级下拉列表按医院等级过滤
- 多条件组合过滤时会同时应用所有条件

### 添加医院
1. 点击"+ 添加新医院"按钮
2. 填写必填信息（医院名称、城市、等级）
3. 可选填其他信息（地址、电话、医生数、床位数、简介）
4. 点击"保存"按钮

### 查看详情
1. 点击医院卡片上的"查看详情"按钮
2. 在详情页面可以看到完整的医院信息
3. 可以直接从详情页面编辑或删除医院

### 编辑医院
1. 在医院卡片上点击"编辑"按钮，或
2. 在详情页面点击"编辑"按钮
3. 修改相关信息
4. 点击"保存"按钮

### 删除医院
1. 在医院卡片上点击"删除"按钮，或
2. 在详情页面点击"删除"按钮
3. 确认删除操作
4. 医院信息将被永久删除

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript (ES6)** - 业务逻辑
- **LocalStorage API** - 数据持久化

## 数据格式

医院对象结构：
```javascript
{
    id: number,              // 唯一标识
    name: string,            // 医院名称
    city: string,            // 城市
    level: string,           // 医院等级
    address: string,         // 详细地址
    phone: string,           // 联系电话
    doctors: number,         // 医生数量
    beds: number,            // 床位数
    description: string      // 医院简介
}
```

## 特色功能

### 1. 实时搜索
输入文字时即时过滤医院，支持模糊匹配

### 2. 多条件组合过滤
可以同时按城市和等级过滤，快速找到目标医院

### 3. 本地数据存储
所有修改都会自动保存到浏览器本地存储，关闭浏览器后数据不会丢失

### 4. 响应式设计
完美适配各种屏幕尺寸，移动设备和桌面都有良好的使用体验

### 5. 用户交互反馈
操作后会显示成功/失败提示，告知用户操作结果
